# ProductScraper
 
- **Run "Program Files\MongoDB\Server\7.0\bin\mongod.exe"**

- **Start MongoDB Compass and connect:**
  - host: localhost
  - port  27017

- **in the project terminal, go to file "productscraper":**
  - ```bash
    cd productscraper
    ```

- **start spider:**
  - ```bash
    scrapy crawl product_scraper
    ```
